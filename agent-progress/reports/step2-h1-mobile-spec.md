# Step2 H1 - 手机端适配规范（V1）

## 1. 断点与栅格
- `<=1200`: 平板/小屏桌面（两栏可降一栏）
- `<=768`: 手机主断点（单栏）
- `<=480`: 小屏手机（按钮/输入区加大触控面积）

## 2. 页面级要求
1. 首页
- 顶部导航折叠为抽屉菜单。
- Banner/轮播高度自适应，主按钮可一键触达。

2. 产品列表/详情
- 列表卡片改两列->一列。
- 详情页图文改纵向，购买按钮固定底部安全区。

3. DIY步骤流
- 步骤条改横向可滚动或顶部简化标签。
- 表单组件统一最小高度 44px，避免误触。

4. 购物车/下单
- 价格汇总与提交按钮固定底部。
- 地址与备注输入区分步展开。

5. 个人中心
- 入口聚合（订单、收藏、工单）为卡片分组。

## 3. 交互与可用性
- 触控目标 >= 44x44。
- 字体最小 14px，关键数值/价格 >= 16px。
- 所有弹窗支持全屏模式（手机端）。
- 输入框获得焦点时避免遮挡（滚动到可见区）。

## 4. 验收清单
- iPhone 12/13/14 视口通过。
- Android 360x800、393x873 视口通过。
- 主流程：登录->DIY->加购->下单可完整执行。
