# 生利达冷冻空调配件DIY商城平台

## 项目概述

生利达冷冻空调配件DIY商城平台是一个专业的B2B+B2C在线商城系统,专注于冷冻空调配件销售和解决方案提供。

## 核心特色

### 1. 智能配套方案系统
- **DIY配套选型器**: 根据客户需求智能推荐配件组合
- **方案可视化**: 3D展示配件组合效果
- **兼容性检测**: 自动检测配件之间的兼容性
- **成本计算器**: 实时计算方案总成本

### 2. 专业技术支持
- **在线技术咨询**: 7×24小时技术支持
- **安装指导视频**: 配件安装教程库
- **故障诊断工具**: AI辅助故障排查
- **知识库**: 冷冻空调专业知识中心

### 3. 一站式服务
- **产品销售**: 压缩机、蒸发器、冷凝器等全系列产品
- **方案定制**: 专业团队定制解决方案
- **安装服务**: 全国联网安装服务
- **售后保障**: 完善的售后服务体系

## 技术栈

### 前端
- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Next.js 14 (服务端渲染)
- Three.js (3D可视化)

### 后端
- Node.js + Express
- PostgreSQL (主数据库)
- Redis (缓存)
- Elasticsearch (搜索引擎)
